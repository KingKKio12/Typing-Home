⌨️ Typing Practice Website
📖 Description

A simple and interactive typing practice website that helps users improve their typing speed and accuracy. It displays random sentences, tracks time, and calculates words per minute (WPM). The site is lightweight, fast, and works offline.

🚀 Features

Random typing text generator

Real-time timer

WPM (Words Per Minute) calculator

Restart with new text

Clean and responsive design

Works without internet

🛠️ How to Use

Open the HTML file in your browser

Start typing the displayed text

Track your time and WPM

Click "New Text" to try again

📂 Installation

No installation needed!
Just download and open the .html file in any web browser.
